# KissKH Android App

A native Android application for streaming Asian dramas and movies from KissKH.

## Features

- **Massive Library**: Access a vast collection of Asian dramas and movies.
- **Watchlist**: Keep track of what you want to watch.
- **History**: Resume playback from where you left off.
- **Android TV Support**: Optimized for the big screen with full remote control navigation.
- **Search**: Easily find your favorite titles.

## Download

[**Download Latest APK**](https://github.com/LazyNinja435/kisskhAndroidApp/releases/latest)

## Installation

1. Download the APK file from the link above.
2. If prompted, enable "Install from Unknown Sources" in your device settings.
3. Open the downloaded file and tap "Install".
